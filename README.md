# Getting Started

### Clone the project:

```bash
git clone https://github.com/ThiagoVRabethge/improved-spork
```

### Add the .env file:

```bash
cd improved-spork
```

```bash
VITE_API_BASE_URL="http://localhost:8000"
```

### Install the dependencies:

```bash
npm install
```

### Run the project locally:

```bash
npm run dev
```
